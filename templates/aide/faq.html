Questions générales

Pour vous connecter :

  1. Allez sur la page de connexion
  2. Saisissez votre nom d'utilisateur et mot de passe
  3. Cliquez sur "Se connecter"

Si vous avez oublié votre mot de passe, contactez votre administrateur.

Pour changer votre mot de passe :

  1. Allez dans votre profil utilisateur
  2. Cliquez sur "Modifier le mot de passe"
  3. Saisissez votre ancien mot de passe
  4. Entrez votre nouveau mot de passe (au moins 8 caractères)
  5. Confirmez le nouveau mot de passe
  6. Cliquez sur "Enregistrer"

Si vous rencontrez des problèmes de connexion :

  • Vérifiez que votre nom d'utilisateur et mot de passe sont corrects
  • Assurez-vous que votre compte n'est pas désactivé
  • Vérifiez votre connexion internet
  • Essayez de vider le cache de votre navigateur
  • Contactez votre administrateur système si le problème persiste
Gestion des clients

Pour ajouter un nouveau client :

  1. Allez dans "Gestion des clients"
  2. Cliquez sur "Nouveau Client"
  3. Remplissez le formulaire avec les informations du client
  4. Cliquez sur "Enregistrer"

Champs obligatoires : Nom complet, téléphone, type de client

Vous pouvez rechercher un client de plusieurs façons :

  • Recherche textuelle : Tapez le nom, téléphone ou email
  • Filtres : Utilisez les filtres par type, statut ou date
  • Recherche par téléphone : Tapez juste les chiffres
Gestion des devis

Pour créer un devis :

  1. Allez dans "Gestion des devis"
  2. Cliquez sur "Nouveau Devis"
  3. Sélectionnez le client
  4. Remplissez l'objet et la description
  5. Ajoutez les lignes d'articles avec quantités et prix
  6. Vérifiez les totaux calculés automatiquement
  7. Cliquez sur "Enregistrer"

Pour imprimer un devis :

  1. Ouvrez le devis que vous voulez imprimer
  2. Cliquez sur le bouton "Imprimer"
  3. Une nouvelle fenêtre s'ouvre avec le devis formaté
  4. Utilisez Ctrl+P pour imprimer ou enregistrer en PDF
Gestion des factures

Pour enregistrer une facture fournisseur :

  1. Allez dans "Gestion des factures"
  2. Cliquez sur "Nouvelle Facture"
  3. Sélectionnez le fournisseur
  4. Saisissez le numéro de facture du fournisseur
  5. Ajoutez les lignes d'articles facturés
  6. Vérifiez les montants et le taux de TVA
  7. Cliquez sur "Enregistrer"

Pour suivre le paiement :

  1. Ouvrez la facture concernée
  2. Utilisez les boutons d'action pour changer le statut :
    • Valider : Facture vérifiée et acceptée
    • Payer : Facture payée
    • Annuler : Facture annulée
Bons de commande

Pour créer un bon de commande d'achat :

  1. Allez dans "Bons de commande"
  2. Cliquez sur "Nouveau Bon de Commande"
  3. Sélectionnez "Achat fournisseur" comme type
  4. Choisissez le fournisseur
  5. Définissez les articles à commander
  6. Imprimez le bon pour le chargé d'achat

Pour imprimer un bon de commande :

  1. Dans la liste des commandes, cliquez sur l'icône d'impression
  2. Une nouvelle fenêtre s'ouvre avec le bon formaté
  3. Le bon s'imprime automatiquement
  4. Vous pouvez aussi utiliser Ctrl+P pour un contrôle manuel

Conseil : Le bon imprimé peut être remis au chargé d'achat pour les achats.

Questions techniques

Oui, l'application est sécurisée :

  • Authentification obligatoire pour tous les utilisateurs
  • Système de permissions par module
  • Chiffrement des données sensibles
  • Sauvegardes automatiques régulières
  • Protection contre les injections SQL

Les sauvegardes sont automatiques, mais vous pouvez :

  1. Aller dans "Paramètres"
  2. Cliquer sur "Sauvegarde"
  3. Télécharger une copie de vos données
  4. Conserver cette copie dans un endroit sûr

Recommandation : Faites une sauvegarde avant toute mise à jour importante.
